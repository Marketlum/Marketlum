# Agreements

Agreements formalize the rules, terms, and commitments between market participants.

## Structure

An agreement consists of:

- **Title** and optional **content** (rich text)
- **Parties** &mdash; two or more agents bound by the agreement
- **Template** &mdash; optional reference to an agreement template
- **File** &mdash; optional attached document

## Hierarchy

Agreements are hierarchical. A master agreement can have child agreements (e.g., annexes, amendments).

## Agreement Templates

Templates provide reusable structures for agreements:

| Type | Description |
|------|-------------|
| **Main Agreement** | The primary contract (e.g., MSA, NDA) |
| **Annex** | Additional terms appended to a main agreement |
| **Schedule** | Specific operational details (e.g., SLA) |
| **Exhibit** | Supporting documentation or specifications |

Templates are also hierarchical, allowing you to model template inheritance.
