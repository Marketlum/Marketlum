# Channels

Channels describe how market participants reach each other and through which mediums exchanges occur.

## Structure

A channel has:

- **Name** and **purpose**
- **Color** &mdash; for visual identification
- **Agent** &mdash; optional owner or operator of the channel

## Hierarchy

Channels are hierarchical. A root channel can have child channels:

```
Direct Sales
└── Field Sales
Online
└── Website
Partner Network
└── VAR Channel
```

## Usage

Channels are referenced by exchanges to indicate *how* a transaction took place. This enables analysis of which channels drive the most activity.
